# 翻译工具访问密码保护实施记录

日期：2026-05-31

## 目标

防止其他人直接访问实时翻译页面或调用后端 API 消耗 OpenAI API Key。日常入口仍保持：

`	ext
https://xcu.ai/trans/
`

## 方案

新增 Cloudflare Pages Middleware：

`	ext
functions/_middleware.js
`

保护范围：

- /trans、/trans/、/trans/*
- /realtime-translation-poc、/realtime-translation-poc.html
- 前端模块 /realtime-translation.mjs、/text-translation.mjs
- OpenAI API 代理 /api/openai/*、/api-proxy/api/openai/*
- Pages 直连根路径 /

未认证浏览器访问页面时会看到密码页。登录成功后设置 HttpOnly + Secure + SameSite=Lax cookie，有效期 30 天。

## Cloudflare Secrets

已写入 Cloudflare Pages 项目 	rans：

- TRANS_ACCESS_PASSWORD：用户访问密码
- TRANS_ACCESS_SECRET：cookie HMAC 签名 secret

密码不写入代码和仓库。

## 验证

新增测试：

`	ext
test/cloudflare-auth-middleware.test.mjs
`

覆盖：

- 未认证访问翻译页面显示密码页
- 未认证 API 返回 401
- 正确密码登录后设置可信浏览器 cookie
- 带可信 cookie 后允许访问受保护路径
