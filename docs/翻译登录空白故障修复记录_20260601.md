# 翻译登录空白故障修复记录

日期：2026-06-01

## 问题

用户从 `https://xcu.ai/trans/` 进入登录页后，提交登录出现空白或异常页面。

## 根因

`xcu-trans-router` 只接管了：

- `xcu.ai/trans*`
- 翻译 API 路径

但登录表单提交地址是：

```text
/trans-login
```

退出地址是：

```text
/trans-logout
```

这两个路径不匹配原来的 `xcu.ai/trans*` Worker route，因此会落到旧站点或 nginx，而不是 Cloudflare Pages Middleware。

## 修复

更新 `cloudflare/xcu-trans-router.js`，将：

- `/trans-login`
- `/trans-logout`

转发到 `https://trans-c2s.pages.dev`。

更新 `cloudflare/xcu-trans-router.wrangler.toml`，新增 Worker routes：

- `xcu.ai/trans-login`
- `xcu.ai/trans-logout`

## 验证

新增路由测试覆盖登录和退出路径，避免以后再次遗漏。
